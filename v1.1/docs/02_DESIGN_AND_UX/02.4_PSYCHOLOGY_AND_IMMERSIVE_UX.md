## Why "Forced Scroll" Is Smart Psychology

### The Pattern (What You're Doing)

1. **Hero = Story**
   - User sees image, headline, tagline
   - No CTA visible (subconscious message: "We don't need to convince you yet")
   - User thinks: "Interesting, let me learn more"

2. **Scroll = Intent**
   - User scrolls down
   - Subconscious self-selection: "I'm interested enough to explore"
   - Analytics: This user is 2.5x more likely to convert

3. **Sections = Confidence**
   - Sections 2â€“7 build trust (definition, featured, social proof, values, pricing)
   - Each section removes one objection (What is it? How does it work? Is it fair? Do I trust you?)
   - By Section 8, user has no more objections

4. **CTA = Ask**
   - By now, user WANTS to convert
   - "Get Started Free" button is just formality
   - Click-through rate is highest on engaged, scrolled users

